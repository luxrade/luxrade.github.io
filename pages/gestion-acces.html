Gestion d'accès (WPF)

Application WPF en C# permettant de gérer les accès aux bâtiments de l’entreprise GSP. Elle inclut la gestion des utilisateurs, l’attribution de badges, le suivi des entrées et sorties et une interface ergonomique basée sur SQL Server.

Technologies utilisées

C#, WPF, SQL Server

Rôle

Conception, développement et tests
